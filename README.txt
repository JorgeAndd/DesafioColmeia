O Programa de busca de professores exibe uma listagem de professores, permitindo uma busca por nome.
Desenvolvido com HTML, CSS e Javascript, com frameworks Parse e Electron.


Instruções para execução:

Execução pelo browser:
	Basta abrir o arquivo app/index.html

Execução com electron:
	Por meio da linha de comando é possível executar:
		electron caminho_para_pasta

Executável
	No repositório do github é possível encontrar um release(zipado) para Windows 64bits
	https://github.com/JorgeAndd/DesafioColmeia/releases
	
	Basta extrair o arquivo e executar BuscaProfessores.exe
